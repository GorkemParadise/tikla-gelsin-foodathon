# TAB Food — Platform Market Share Dashboards

Interactive HTML dashboards showing online food delivery platform market shares (Yemeksepeti, Trendyol Food, Getir, Tıkla Gelsin) by district for TAB Food restaurants (Burger King, Popeyes, Subway, Arby's, Sbarro, Usta Dönerci, Usta Pideci).

## Data

Monthly estimated order figures, 2024–2025 model. Most of the data was obtained from TAB Foods, and missing data were filled in using model-based methods. (March 2026).
